Kenapa python?

Kenapa Python?

Banyak kelebihan dan kekurangan di berbagai bahasa pemrograman, jadi saya tidak akan menjelaskan secara detail satu-per-satu bahasa pemrograman, silakan browsing. Tetapi saya akan menjelaskan kelebihan dan kekurangan bahasa Python.

Kelebihan bahasa Python

  1. Dukungan komunitas.
    Komunitas python adalah komunitas yang sangat aktif mengembangkan python sehingga menjadi bahasa yang sangat handal.

  2. Bahasa pemrograman yang populer.
    Dengan kepopuleran, sudah pasti banyak yang menggunakan. Ketika ada error atau masalah tinggal cari penyelesaiannya melalui browsing atau bertanya dalam suatu grup atau komunitas.

  3. Kode Python mudah ditulis dan mudah dibaca.
    Sehingga lebih mudah diperbaiki kalau ada kesalahan, dan juga mudah untuk dipelihara.

  4. Kode program lebih efisien.
    Jika di bahasa lain butuh 5 baris di Python cukup 1 baris.

  5. Python merupakan bahasa multifungsi.
    Dengan python kita bisa melakukan berbagai hal mulai dari memproses teks, membuat website, membuat program jaringan, robotika, data mining, sampai dengan kecerdasan buatan. Kita bisa membuat aplikasi berbasis desktop maupun berbasis smartphone.

  6. Python kaya akan dukungan library(pustaka) standar.
    Tersedia banyak sekali modul-modul dan ekstensi program yang sudah siap Anda pakai untuk membuat program sesuai kebutuhan Anda.

  7. Python bisa berinteraksi dengan bahasa lain.
    Kode python bisa memanggil bahasa C, C++, dan sebaliknya juga bisa dipanggil dari bahasa lain.

  8. Freeware.
    Python bersifat gratis atau bebas (free) dan open source, meskipun digunakan untuk kepentingan komersial.

Kekurangan bahasa Python

Selain kelebihan sudah pasti ada kekurangannya, tinggal gimana caranya menutup kekurangan dengan kelebihan tersebut.

Python adalah bahasa interpreter. Kekurangan python dibanding bahasa lain yang tidak menggunakan compiler adalah sedikit lebih lambat pada saat dijalankan. Tapi hal inipun sangat bersifat relatif. Tergantung dari besar ukuran program yang dibuat.

Untuk program besar yang membutuhkan kecepatan pemrosesan tinggi mungkin Python kalah cepat dari bahasa C, tapi untuk hal selain itu Python lebih mudah dan lebih baik dari bahasa lain.

Selain itu, kode sumber sekarang sudah dioptimasi menggunakan bahasa C, sehingga kecepatannya juga sudah sangat mendekati kecepatan bahasa C.

Spesifikasi komputer juga sekarang ini sudah semakin tinggi sehingga bisa memproses program dengan cepat, sehingga sering kali ini tidak menjadi hal penting dan bisa diabaikan.
