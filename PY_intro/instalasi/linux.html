Linux

Instalasi Python di Linux

Untuk beberapa distro dari OS Linux secara default sudah terinstall Python.

Silakan di cek dulu.

Silakan buka Terminal, lalu ketikan python - V atau python3 - V lalu enter.

Akan muncul versi Python.

Jika tidak ada atau versi Python bukan versi 3.x. Silakan ikuti cara install dibawah ini.

Package Management Apt

Linux dengan distro Debian beserta variannya, Ubuntu, Kali, Mint, dll.

  1. Buka Terminal.
  2. Ketik sudo apt install python3 lalu enter.
  3. Ketik y untuk konfirmasi penginstallan.
  4. Jika sudah selseai, ketik python3 -V akan muncul versi Python yang terinstall.

Package Management Yum

Linux dengan distro Red Hat beserta variannya, Fedora, CentOS, Oracle Linux, dll.

  1. Buka Terminal.
  2. Ketik sudo yum install python3 lalu enter.
  3. Ketik y untuk konfirmasi penginstallan.
  4. Jika sudah selseai, ketik python3 -V akan muncul versi Python yang terinstall.

Unix atau Linux

Jika cara diatas enggak bisa. Pakai cara manual dibawah, untuk Unik atau Linux.

  1. Download https://www.python.org/ftp/python/3.8.3/Python-3.8.3.tgz
  2. Ekstrak dengan tar zxvf Python-3.8.3.tgz enter
  3. Jalankan ./configure script.
  4. make enter.
  5. make install enter.
  6. Selesai

Mode Interpreter atau Interaktif

Modus interaktif/console/Interpreter adalah tool untuk menjalankan perintah-perintah atau fungsi-fungsi python. Modus ini biasanya digunakan untuk percobaan dan latihan.

Tanda >>> primary prompt, artinya interpreter Python telah siap untuk menerima perintah.

Tanda ... secondary prompt atau sub prompt, mucul saat membuat blok kode dan menulis perintah tunggal dalam beberapa baris.

Contoh :

>>>print('Hello World!!!') 
Hello World!!! 
>>>for i in range(2) :
...    print('Ya') 
... 
Ya
Ya
>>>

Untuk keluar dari mode interaktif, menggunakan tombol kombinasi Control+C kemudian Control+D. Atau menggunakan perintah exit() lalu enter.

Menjalankan file .py

Dengan command python3 dir/file.py.

Contoh menjalankan file hello.py di direktori home/RP/

  1. Buka Terminal.
  2. Masuk ke direktori cd /home/RP/ enter.
  3. Lihat file dan direktori saat ini ls enter.
  4. Jalankan file python3 hello.py.
  5. Selesai.
