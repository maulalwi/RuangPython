Instalasi Python di Android

Instalasi Python di Android memakai aplikasi pihak ketiga(bukan aplikais official dari Python).

Dengan Python yang diinstall di Windows atau Linux apakah sama? Sama saja, hanya beda di cara pemakaiannya.

Install Pydroid 3 dari playstore

  1. Download Pydroid 3 di playstore https://play.google.com/store/apps/details?id=ru.iiec.pydroid3 , install seperti biasa, lalu buka.
  2. Silakan tunggu instalasinya.
  3. Setelah selesai akan muncul tampilan utama text editor Pydroid 3.

Tampilan awal Pydroid 3

Penjelasan :

Fokus diarea yg diberi lingkaran dan nomor.

  1. Area untuk menulis Kode Python.
  2. Untuk menjalankan kode Python.
  3. Klik icon folder, akan muncul menu New, Open, Save, Save as, Close, Recent.

Menu disebelah kiri

Penjelasan :

Fokus di area dalam kotak.

1. Mode Interpreter atau Interaktif

Modus interaktif/console/Interpreter adalah tool untuk menjalankan perintah-perintah atau fungsi-fungsi python. Modus ini biasanya digunakan untuk percobaan dan latihan.

Tanda >>> primary prompt, artinya interpreter Python telah siap untuk menerima perintah.

Tanda ... secondary prompt atau sub prompt, mucul saat membuat blok kode dan menulis perintah tunggal dalam beberapa baris.

Contoh :

>>>print('Hello World!!!') 
Hello World!!! 
>>>for i in range(2) :
...    print('Ya') 
... 
Ya
Ya
>>>

Untuk keluar dari mode interaktif, menggunakan tombol kombinasi Control+C kemudian Control+D. Atau menggunakan perintah exit() lalu enter.

2. Mode Terminal

Didalam terminal Pydroid3 hanya ada beberapa perintah dasar Linux saja, antara lain :

  1. python

    • Interpreter -> python
    • Run file -> python dir/file.py

  2. pip
    Management paket python.

    • List -> pip list
    • Search -> pip search nama_paket
    • Install -> pip install nama_paket
    • Uninstall -> pip uninstall nama_paket

  3. ls
    Menampilkan file dan folder dalam direktori.

  4. cd
    Merubah lokasi dir.

  5. Pindah -> cd dir/

  6. Kembali -> cd ../

  7. mkdir
    Membuat folder baru -> mkdir nama_folder

  8. cp

  9. Copy file -> cp file_lama file_baru

  10. Copy folder -> cp -r folder_lama folder_baru

  11. mv
    Untuk memindahkan, juga bisa untuk merubah nama file atau folder.

  12. Move file -> mv file_lama file_baru
  13. Move folder -> mv folder_lama folder_baru
