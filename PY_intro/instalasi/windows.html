Windows

instalasi Python di Windows

  1. Buka browser, kunjungi http://www.python.org/downloads/windows/ download versi terbaru 3.x atau melalui direct link https://www.python.org/ftp/python/3.8.3/python-3.8.3-amd64.exe

  2. Buka (klik 2x) file installer python yang baru saja di download.

  3. Menu instalasi Python.

  4. Abaikan versi di screenshot. Yang terpenting centang di nomor 1 (Add Python to PATH), lalu Install now.

  5. Ikuti langkah instalasi sampai selesai.

  6. Jika instalasi selesai akan muncul pop up Setup was successful. Itu berarti instalasi berhasil.

Mode Interpreter atau Interaktif

Modus interaktif/console/Interpreter adalah tool untuk menjalankan perintah-perintah atau fungsi-fungsi python. Modus ini biasanya digunakan untuk percobaan dan latihan.

  1. Buka CMD (Dimenu search windows ketik CMD, lalu klik).
  2. Di CMD, ketikan python lalu enter.
  3. Akan muncul versi Python yang dipakai.

Tanda >>> yang artinya mode Interpreter Python siap menerima kode.

Tanda ... secondary prompt atau sub prompt, mucul saat membuat blok kode dan menulis perintah tunggal dalam beberapa baris.

Contoh :

>>>print('Hello World!!!') 
Hello World!!! 
>>>for i in range(2) :
...    print('Ya') 
... 
Ya
Ya
>>>

Untuk keluar dari mode interaktif, menggunakan tombol kombinasi Control+C kemudian Control+D. Atau menggunakan perintah exit() lalu enter.

Melalui File

Menjalankan kode program melalui file.

  1. Buat folder RP atau boleh dinamai lain, di C:\. Lalu buka folder RP.
  2. buat file berextensi .py di folder RP. Misal hello.py
  3. Jadi, alamat file lengkapnya C:\RP\hello.py.
  4. Buka file hello.py dengan text editor, kemudian ketikkan print("Hello World!"). Save.
  5. Buka CMD.
  6. Pindah ke direktori tadi, dengan perintah cd C:\RP\ lalu enter.
  7. Ketikan ls lalu enter, untuk melihat nama file dan direktori.
  8. Ketikkan python hello.py lalu enter. Harus sesuai dengan nama filenya, kalau salah akan muncul error.
  9. Kode program berjalan.

Intinya di, python lokasi\file\name.py lalu enter.
